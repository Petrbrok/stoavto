# Запуск на Synology

1. Скопируйте папку проекта на NAS, например в `/volume1/docker/stoavto`. Папки `node_modules` и `.next` копировать не нужно.
2. Создайте `.env` рядом с `compose.yaml` по образцу `.env.example` и замените секреты и домен.
3. Скопируйте папку `data` вместе с `database.db`. Она подключается в контейнер как `/app/data` и хранит базу и загрузки между пересборками.
4. Дайте пользователю контейнера UID `1001` права чтения и записи на папку `data`. Через SSH: `sudo chown -R 1001:1001 /volume1/docker/stoavto/data`.
5. В Container Manager откройте **Проект** -> **Создать**, укажите имя `stoavto` и путь `/volume1/docker/stoavto`.
6. Выберите источник **Создать compose.yaml** или существующий файл, проверьте содержимое `compose.yaml`, нажмите **Далее** -> **Готово**.
7. Дождитесь статуса проекта **Работает** и откройте `http://IP_ВАШЕГО_NAS:3000`.

Обновление: замените файлы проекта, затем в Container Manager выберите проект `stoavto` -> **Действие** -> **Собрать**. Папку `data` не удаляйте.
